# راهنمای کامل سایت (برای کسی که کدنویسی بلد نیست)

## هر روز چطور سایت رو آپدیت کنم؟

تا وقتی کسی کد رو تغییر نده، سایت خودش کار میکنه و نیازی به هیچ کاری نداری.

**وقتی خواستی چیزی توی سایت عوض کنی:**

```
1. فایل مورد نظر رو باز کن (مثلاً app/page.js برای صفحه اصلی)
2. متن یا محتوا رو عوض کن
3. توی ترمینال این سه تا دستور رو به ترتیب بزن:

   git add .
   git commit -m "توضیح بده چی عوض کردی"
   git push

4. برو به Vercel.com - اونجا میبینی سایت داره آپدیت میشه (۲-۳ دقیقه)
5. تموم! سایت آپدیت شد 🎉
```

---

## آموزش کامل گام به گام

### ۱- اولین بار که میخوای سایت رو بذاری روی سرور

#### قدم ۱: ثبت‌نام در سرویس‌ها
1. برو به **vercel.com** و با حساب گیت‌هاب ثبت‌نام کن
2. برو به **neon.tech** و ثبت‌نام کن (برای دیتابیس)
3. برو به **uploadthing.com** و ثبت‌نام کن (برای عکس‌ها)

#### قدم ۲: ساخت دیتابیس در Neon
1. توی Neon یه پروژه جدید بساز (region: US East)
2. از صفحه اطلاعات، **Connection string** رو کپی کن (چیزی شبیه `postgresql://...`)
3. این رو بعداً توی Vercel میذاریم

#### قدم ۳: ساخت API Key برای Uploadthing
1. توی Uploadthing یه اپ جدید بساز
2. دو تا کلید میگیره: `UPLOADTHING_SECRET` و `UPLOADTHING_APP_ID`
3. اینا رو هم بعداً توی Vercel میذاریم

#### قدم ۴: دیپلوی روی Vercel
1. توی Vercel دکمه **Add New → Project** رو بزن
2. ریپازیتوری گیت‌هاب رو انتخاب کن
3. توی صفحه تنظیمات، پایینش **Environment Variables** رو باز کن و اینا رو اضافه کن:

| نام | مقدار |
|-----|-------|
| `DATABASE_URL` | اون رشته PostgreSQL از نئون |
| `NEXTAUTH_SECRET` | یه متن تصادفی مثل `my-super-secret-key-12345` |
| `NEXTAUTH_URL` | آدرس سایتت (مثلاً `https://student-market.vercel.app`) |
| `UPLOADTHING_SECRET` | کلید اپلودتینگ |
| `UPLOADTHING_APP_ID` | آیدی اپلودتینگ |

4. دکمه **Deploy** رو بزن
5. **حدود ۳ دقیقه** صبر کن تا سایت آماده بشه 🚀

---

### ۲- عوض کردن متن‌های سایت (مهم‌ترین بخش)

#### عوض کردن متن فارسی/انگلیسی/ایتالیایی
فایل: `lib/i18n.js`

مثلاً اگه بخوای اسم سایت رو عوض کنی:
- توی فایل `lib/i18n.js` رو باز کن
- دنبال `siteName: "خانه دانشجویی مسینا"` بگرد
- متن رو عوض کن به چیزی که میخوای
- تغییر تو هر سه زبان (fa/en/it) رو اعمال کن

#### عوض کردن متن توی صفحه اصلی
فایل: `app/page.js`

#### عوض کردن متن توی نوار بالایی
فایل: `app/components/Navbar.js`

#### عوض کردن متن توی صفحه جزئیات محصول
فایل: `app/products/[id]/page.js`

---

### ۳- مدیریت محتوا

#### اضافه کردن دسته‌بندی جدید
- برو به آدرس `yoursite.com/admin/categories`
- اسم و آیکون دسته جدید رو انتخاب کن و ذخیره کن

#### مدیریت پیام‌ها
- برو به `yoursite.com/admin/messages`
- همه مکالمات کاربران رو میبینی

#### تغییر رنگ‌های سایت
- برو به `yoursite.com/settings/theme`
- رنگ‌ها رو عوض کن و ذخیره کن

---

### ۴- سوالات متداول

**چطور کاربر جدید بسازم؟**
اگه میخوای یه نفر ادمین بشه، باید مستقیم توی دیتابیس توی Neon
توی جدول User فیلد isAdmin رو رو true بذاری.

**چرا عکس‌ها آپلود نمیشه؟**
حتماً کلیدهای Uploadthing رو درست توی Vercel تنظیم کرده باش.

**چطور بفهمم سایت بالا هست؟**
برو به Vercel.com → پروژه رو باز کن → میبینی که "Ready" نوشته یا نه.

**اگه سایت کرش کرد چیکار کنم؟**
توی Vercel → پروژه → Function Logs رو نگاه کن. خطاها اونجا نوشته میشه.

---

### ۵- آپدیت کردن کد بعد از اولین دیپلوی

کافیه از ترمینال سه تا دستور بزنی:

```bash
git add .
git commit -m "توضیح بده چی عوض کردی مثلاً: تغییر رنگ دکمه‌ها"
git push
```

همین! Vercel خودکار تشخیص میده و سایت رو آپدیت میکنه.

اگه از GitHub Desktop استفاده میکنی، حتی راحت‌تره:
۱. فایل رو عوض کن
۲. توی GitHub Desktop تغییرات رو ببین
۳. یه توضیح بنویس و Commit و Push بزن
